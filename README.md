# NestHR — HR Management System

> A complete HR software for small and medium businesses. Manage your employees, attendance, payroll, leaves, and more — from both a **web browser** and a **mobile app (iOS & Android)**.

---

## What is NestHR?

NestHR helps HR managers and business owners run their company's people operations without needing spreadsheets or paper files. Think of it as your digital HR office — everything about your employees, their salary, leaves, and attendance, all in one place.

There are **two versions**:
- **Web App** — Open in any browser on your computer (like a website)
- **Mobile App** — Install on your phone (Android / iPhone)

## ❌ What is NOT Yet Built (Still Pending)

### Web App — Missing Features

#### Reports (Pending — Show "Coming Soon")
These reports are listed in the app but clicking them does nothing yet:

| Report Name | Why HR Needs It |
|---|---|
| PT Register Report | Proof of Professional Tax deducted — needed for compliance |
| LWF Register Report | Labour Welfare Fund record — required by law in some states |
| Loan Report | Summary of all employee loans and repayments |
| Bonus Report | Yearly bonus details per employee |
| Overtime Summary Report | How many extra hours employees worked and what to pay |
| Compliance Report | Overall statutory compliance health check |
| Early Going Summary | Which employees left office early and by how much |
| Excess Break Summary | Which employees took longer breaks than allowed |
| PF Challan ECR Report | File to upload on EPFO portal every month — very important |
| ESIC Challan Report | File to upload on ESIC portal every month — very important |
| Attendance In/Out Vertical | Different format of punch in/out report |
| Locationwise Report | Attendance breakdown by office location/branch |
| Punch Log Detail Report | Full raw punch data from biometric machine |
| Attendance History | Long-term attendance record view |

#### Other Missing Features

| What's Missing | What HR Actually Needs |
|---|---|
| **Employee Document Storage** | No place to upload/store documents like Aadhaar card, PAN card, offer letter copies, relieving letters for each employee. |
| **Overtime Management** | You can set shift timings but there is no screen to actually record or approve overtime hours for employees. |
| **Asset Management** | No way to track company assets (laptop, phone, access card) given to employees. |
| **Training & Learning Module** | No place to assign training courses, track certifications, or record skill development of employees. |
| **Notification Center (Bell Icon)** | Push notification infrastructure exists in the code but there is no bell icon in the app header where HR can see all recent alerts. |
| **Payroll Tax Computation (TDS)** | The system calculates PF and ESIC but there's no TDS (income tax) deduction calculation built in. |
| **Multi-Branch / Location Management** | Company profile is single-location only. If your company has offices in multiple cities, you can't manage them separately. |

### Mobile App — Missing Features

| What's Missing | What HR Actually Needs |
|---|---|
| **Biometric Device Management** | The web app lets you manage fingerprint/NFC machines, but the mobile app has no such screen. |
| **NFC Card Assignment** | Assigning access cards to employees only works on the web, not on mobile. |
| **Payroll Settings / Deduction Rules** | You can't change late deduction rules from the mobile app. |
| **Employee Self-Service on Mobile** | Right now the mobile app is only for the HR admin. Employees cannot log in to the mobile app to see their own payslip or apply for leave from their phone. |
| **Profile Photo Upload** | Cannot take a photo or upload a profile picture from the mobile app. |
| **Offline Mode** | If the internet is slow or gone, the app shows nothing. It should cache some data so basic viewing still works offline. |
| **Forgot Password Flow** | No "Forgot Password" option on the mobile login screen. |
| **Dark Mode** | The app only has a white/light theme. No dark mode option. |
| **Push Notification UI** | The backend can send push notifications but the mobile app has no in-app notification center to view them. |
| **Document Viewer** | Cannot view employee documents or payslip PDFs inside the mobile app. |
| **Exit Management on Mobile** | The web app has a full exit/resignation flow, but there is no mobile screen for it. |
| **2FA on Mobile** | Two-factor authentication only works on the web login. Mobile login does not ask for a 2FA code yet. |

## Tech Stack (For Developers)

| Layer | Technology |
|---|---|
| Backend (Server) | Node.js + Express.js |
| Database | MongoDB |
| Web Frontend | React + Vite + TypeScript + Tailwind CSS |

| Email | Nodemailer (SMTP) |

## How to Run Locally

### Backend
```bash
cd backend
npm install
npm run dev
```

### Web App
```bash
cd frontend
npm install
npm run dev
```

### Mobile App
```bash
cd app/NestHR
npm install
npx react-native run-ios     # For iPhone simulator
npx react-native run-android # For Android emulator
```
